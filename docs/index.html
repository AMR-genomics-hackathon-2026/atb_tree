How it works

01
Fetch AMRrules
Download species-specific intrinsic gene definitions from AMRrules
02
Fetch AMRFinderPlus
Download ~691 MB of AMRFinderPlus results from OSF (osf.io/ck7st)
03
Filter & aggregate
Remove intrinsic genes using Scope=core + AMRrules species-aware lookup
04
Build metadata
Merge with GTDB taxonomy; produce summary + binary presence-absence matrix
